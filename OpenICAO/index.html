Welcome to OpenICAO!

OpenICAO simply returns the closest weather ICAO station to a given latitude and longitude.

Example: http://api.openweather.pw/Search/?lat=-90&lngt=-180


You can also query NOAA for current weather data with latitude and longitude or ICAO. This will return an XML formatted string of the last 24 hours of observation.

Example: http://api.openweather.pw/Query/?lat=-90&lngt=-180

Example: http://api.openweather.pw/Query/?icao=NZSP



All data returned from the ICAO search portion of this site (http://api.openweather.pw/Search/) is under the BSD 3-Clause License (available here), and is free of use. We do not make claims that the data is accurate.

Weather data returned from the query section of this site (http://api.openweather.pw/Query/) is pulled directly from NOAA.
